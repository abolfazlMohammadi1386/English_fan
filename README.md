# English Fan

سیستم مدیریت آموزشگاه زبان انگلیسی

## Project Description

English Fan is a web-based language institute management system built with Laravel and Filament Admin Panel.

Current Features:

* Admin Dashboard
* User Management
* Authentication System
* Role & Permission System (Planned)
* Student Management (Planned)
* Teacher Management (Planned)
* Course Management (Planned)
* Class Scheduling (Planned)
* Attendance Management (Planned)
* Financial Management (Planned)

## Technology Stack

* Laravel 12
* PHP 8.2
* Filament 4
* MySQL
* Git & GitHub

## Installation

```bash
git clone https://github.com/abolfazlMohammadi1386/English_fan.git

cd English_fan

composer install

cp .env.example .env

php artisan key:generate

php artisan migrate

php artisan serve
```

## Admin Panel

```
http://127.0.0.1:8000/admin
```
